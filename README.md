# 📖 Overview

AutonoSci AI is an end-to-end Autonomous AI Data Scientist platform designed to automate the complete data science workflow.

Instead of manually performing data cleaning, exploratory data analysis, feature engineering, model selection, explainability, reporting, and business interpretation, AutonoSci AI intelligently performs these tasks with minimal user interaction.

The goal is to make advanced data science accessible through a single intelligent platform.

---

# 🎯 Vision

AutonoSci AI aims to become an AI-powered data science assistant capable of:

- Cleaning messy datasets
- Performing automated EDA
- Engineering useful features
- Training multiple ML models
- Selecting the best-performing model
- Explaining predictions using SHAP
- Generating executive business insights
- Producing professional PDF reports
- Assisting users through an AI Copilot
